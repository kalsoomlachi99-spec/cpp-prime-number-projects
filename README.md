# Prime Number Projects in C++

## Overview

This repository contains a collection of C++ programs related to prime numbers. The purpose of these projects is to strengthen programming fundamentals, logical thinking, and problem-solving skills.

---

## Projects Included

### 1. Prime Number Checker
Checks whether a given number is prime or not.

### 2. Prime Number Checker Using Functions
Implements prime number checking using user-defined functions.

### 3. Sum of Prime Numbers from 2 to N
Calculates the sum of all prime numbers within a given range.
how many prime numbers exist within a specified range.
